# 8-bit ALU

## Description

This project implements a simple 8-bit Arithmetic Logic Unit (ALU).

The ALU supports:
- Addition
- Subtraction
- AND
- OR
- XOR

## Inputs

- ui_in → Operand A
- uio_in → Operand B + Opcode

## Output

- uo_out → Result

## Applications

- Processor design
- FPGA projects
- Digital electronics learning
- Tiny Tapeout educational project
